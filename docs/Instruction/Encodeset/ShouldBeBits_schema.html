Instruction.Encodeset.ShouldBeBits object

Deprecated

This has been deprecated in favor of the should_be_mask property of Bits and Fields.

A ShouldBeBits object represents a group of one or more bits in an Encodeset. The range property states how many bits are present and where they are in the Encodeset. The value given is what the individual bits should hold, architecturally, but the actual value present may differ.
