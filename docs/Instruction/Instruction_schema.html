Instruction.Instruction object

An Instruction object defines an encoding that has a name (for example, ADD or SYS) and an operation. Most instances of Instruction also have an assembly representation and a set of user-defined properties.

If an Instruction has children, each is of one of three types:

  • Instruction children use a subset of the parent Instruction encoding and may define a different operation and assembly.
  • InstructionAlias children use the same encoding and operation as the parent Instruction, but define different assembly.
  • InstructionInstance children define instances of the parent Instruction by associating additional metadata with it.
