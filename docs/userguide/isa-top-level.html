Instruction Set Architecture

This guide and its linked sub-pages introduce the AARCHMRS Instruction Set Architecture (ISA) and its data model.

An AARCHMRS ISA Instruction contains three key things:

  • InstructionSets - bit-patterns that define the encoding, opcode, and fields for each instruction.
  • Assembly - the assembly structure for each instruction.
  • Operations - the ASL pseudocode and descriptions for one or more instructions.

AARCHMRS ISA terminology distinguishes an instruction from an operation. An instruction is a bit-pattern that can contain named fields and specified bits. An operation is the behavior performed when an instruction is executed. An operation can have more than one instruction associated with it.

See the ISA Data Model User Guide for information on how the above things are represented in the data.
