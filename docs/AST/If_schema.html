AST.If object

A model representing an ASL conditional statement. Models the following structures:

  • Simple If statements
if a == '00' then
    return '00';
end
  • Nested If blocks
if a == '00' then
    return '00';
elsif a == '01' then
    return '01';
else
    return '11';
end
