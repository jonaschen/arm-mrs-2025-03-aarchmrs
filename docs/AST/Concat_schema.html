AST.Concat object

A model representing a concatenation of multiple values:

  • [REG[31:0], REG[63:32]]
  • PSTATE.[D, A, I, F] which is a shorthand for [PSTATE.D, PSTATE.A, PSTATE.I, PSTATE.F]
