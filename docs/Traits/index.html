Traits

Traits are purpose-specific additional schema elements. The name indicates which schema element it affects. For example, the HasCondition trait is used within the Condition schema element.

  • HasCondition
  • HasConstraints
  • HasDescription
  • HasFieldAccess
  • HasIndexes
