Values.EquationValue object

A EquationValue provides an infrastructure that defines a value given an equation in the format (<value>)[<slice>].

For example ((n * 2) + x)[3:0], where the values of n or x can be defined by an implementation, and [3:0] provides an indication of the size (4-bits) and slice (bits 3:0) of the value after evaluation.

slice◊ Rangeset

This represent the [3:0] part of the EquationValue.

The slice can start from any position (for example [3:1]) or be non-continuous (for example [3:2, 0]) and the evaluation of the value must respect the slice.

See Rangeset for more information.
value◊
  • string

This represents the (n * 2) + x (note without leading parenthesis) part of the EquationValue. The value stores the internal part of the equation.
