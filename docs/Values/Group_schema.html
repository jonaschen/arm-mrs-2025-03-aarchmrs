Values.Group object

A Group allows a value to be represented by the concatenation of different elements.

For example, 0b00, 'xx10', and foo[2:0] can be concatenated using the : symbol as 0b00:'xx10':foo[2:0].

If foo is set to 12 then foo's binary representation is '1100' and the [2:0] slice of that is '100'. This makes the Group example 0b00:'xx10':foo[2:0] represent the bitstring '00xx10100'.
