The current domain (possible values) of this Parameters.Boolean.

By default this is set to both TRUE and FALSE, meaning that the value of this Parameters.Boolean is in an unknown state, and can be set to either TRUE or FALSE directly (by writing a single value to the parameter) or indirectly (through a constraint that requires a particular value).
